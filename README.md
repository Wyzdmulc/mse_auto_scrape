# Malawi Stock Exchange Daily Scraper

This script automatically fetches daily stock data from the [Malawi Stock Exchange](https://mse.co.mw),
stores it in a local SQLite database and Excel file, and sends email alerts.

## Features
- Daily automated run at 16:00
- Stores data in SQLite & Excel
- Logs all activities
- Sends email notifications for success/failure
